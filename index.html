📖 Panduan Setup Google Sheets

① Buat Google Spreadsheet baru
Buka sheets.google.com → buat spreadsheet baru.
Ganti nama Sheet1 menjadi: Hasil LJK
② Buka Apps Script
Di spreadsheet → menu Extensions → Apps Script.
Hapus kode default, lalu paste isi file ljk-apps-script.gs.
③ Deploy sebagai Web App
Klik Deploy → New deployment
  • Type: Web app
  • Execute as: Me
  • Who has access: Anyone
→ Klik Deploy → izinkan akses → copy URL.
④ Hubungkan ke LJK Scanner
Paste URL di kolom sebelah kiri → Simpan URL → Test Koneksi.
Selesai! Setiap klik 💾 Simpan Hasil otomatis terkirim ke Sheets. ✅
